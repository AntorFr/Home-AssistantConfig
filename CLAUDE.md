# CLAUDE.md

Ce dépôt suit les mêmes instructions que GitHub Copilot. **Lis et applique** le fichier suivant pour tout travail sur ce dépôt :

@.github/copilot-instructions.md

## Rappel des points essentiels

- **Avant toute modification de configuration**, consulter le dossier `docs/` pour le contexte fonctionnel : commencer par `docs/CONVENTIONS.md`, puis `docs/index.md`, puis les pages métier concernées (`docs/pieces/`, `docs/routines/`, `docs/fonctions/`).
- **Quand une évolution de configuration modifie un comportement de la maison**, mettre à jour la documentation dans `docs/` dans le même changement, en respectant strictement `docs/CONVENTIONS.md` (style, structure, liens croisés, balises `<!-- source: ... -->`, rigueur factuelle).
